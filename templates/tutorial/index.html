1. Buka menu Klasifikasi pada navbar di atas.

2. Pilih gambar daun cabai dari direktori file yang tersedia di perangkat Anda.

3. Klik tombol "Klasifikasi", lalu sistem akan memproses gambar dan menampilkan hasil diagnosis.

4. Selesai! Anda dapat mencoba gambar lainnya kapan saja.
